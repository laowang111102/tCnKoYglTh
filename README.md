# 图书信息管理系统 python231

本项目是一款基于 Python 与 PyQt5 的图书信息管理系统，旨在帮助用户高效管理图书数据。

## 技术栈

- Python
- PyQt5

## 功能模块

- 显示图书表中所有图书的相关信息
- 插入图书信息到指定位置
- 删除指定位置的图书
- 实现图书信息去重
- 实现图书信息的模糊查询
- 批量修改图书信息
- 登陆注册系统
- 权限管理系统

## 系统角色

系统主要分为以下角色：
- 系统管理员
- 普通用户

## 运行环境

- 操作系统：Windows、Linux、macOS
- Python 环境：Python 3.x

1. 确保计算机已安装 Python 3.x 和 PyQt5。
2. 克隆或下载项目源代码到本地。
3. 执行 `main.py` 启动程序。
